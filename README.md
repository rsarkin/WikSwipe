# 📘 WikSwipe

> Swipe through knowledge like never before.

## 🚀 Overview
**WikSwipe**is an interactive way to explore Wikipedia content using a swipe-based interface. Instead of traditional scrolling, users can discover information quickly by swiping through articles.

It’s designed for fast learning, curiosity driven exploration, and a modern user experience.



 ✨ Features

- 📱 Swipe-based navigation
- 🔍 Real-time Wikipedia content
- ⚡ Fast and minimal UI
- 🧠 Smart/random article discovery
- 📚 Quick learning experience
